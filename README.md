# axe for the frozen sea

> "a book must be the axe for the frozen sea inside us."    - *franz kafka*

## description
mern app that allows users to: 
* search for books 
* save books to favorites
* delete books from favorites

## tech
* react
* node.js
* express.js
* mongodb
* mongoose
* grommet
* google books api
